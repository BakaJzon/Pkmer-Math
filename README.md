## 简介
基于Obsidian建立公开共享的数学知识库！
本库致力于以通俗易懂的语言梳理高等数学的主要内容, 并将其整理为符合 Obsidian 风格的 Markdown 文件. 在内容呈现上, 充分利用了 Obsidian 的特色语法. 由于开发者精力有限, 如有内容疏漏/错误 欢迎补充和指出. 希望共同开发的场合, 请参阅文档.
