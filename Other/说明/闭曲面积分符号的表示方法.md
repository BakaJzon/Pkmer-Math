由于Obsidian不支持二重曲面积分符号`\oiint`的渲染, 因此下面提供一种折中的方式, 可以在Obsidian中渲染出二重积分的符号 
```
\int\kern{-17mu}{\unicode{x25CB}}\kern{-20mu}\int_{C}
```
尽管可以在Obsidian中使用, 这个方法在网页端仍旧无法正常显示

---
## 2026-01-20

### 一个新的方案: 
```
\displaystyle\int\hspace{-0.95em}\bigcirc\hspace{-1em}\int
```

现在使用Extended MathJax插件, 在 根目录下`preamble.sty`文件中定义以下规则: 
```
\newcommand{\oiint}{\displaystyle\int\hspace{-0.95em}\bigcirc\hspace{-1em}\int}
```
即可在文中正常使用`\oiint`来方便调用: $\oiint$ 


同理, 四维空间中的三维闭超曲面积分(可能不太常用): 
```
\displaystyle\int\hspace{-0.85em}\int\hspace{-1.05em}\bigcirc\hspace{-1.15em}\int
```
可定义如下规则: 
```
\newcommand{\oiiint}{\displaystyle\int\hspace{-0.85em}\int\hspace{-1.05em}\bigcirc\hspace{-1.15em}\int}
```
效果: $\oiiint$
