省流
作用: 用一些简单的函数近似代替复杂的函数表达

$$
f( x ) = f ( x _ { 0 } ) + f ^ { \prime } ( x _ { 0 } ) + \frac { f ^ { \prime \prime} ( x _ { 0 } ) } { 2 ! } ( x - x _ { 0 } ) ^ { 2 } + \cdots + \frac { f ^ {(n)} ( x _ { 0 } ) } { n ! } ( x - x _ { 0 } ) ^ n
$$
写成通项形式: 
$$
f(x)=\sum_{n=0}^{n\to\infty} \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n
$$
